# archive-mk-index

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的存储与索引目录。

## 项目简介

`archive-mk-index` 旨在收集、整理并公开存放各类独立的 HTML 页面。这些页面可能来自不同时期、不同主题或不同来源，但均以原始 HTML 格式保留，便于直接浏览与存档。仓库本身不提供内容筛选或推荐，仅作为一份可访问的网页集合。

## 目录说明

```
.
├── index.html          # 主索引页面，列出所有归档页面的链接
├── pages/              # 存放各独立 HTML 页面的文件夹
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源文件（如样式、脚本、图片等）
    ├── style.css
    └── script.js
```

- `index.html`：根目录下的主入口，包含所有归档页面的目录列表。
- `pages/`：每个归档的 HTML 页面以单独文件存放，文件名保持唯一性。
- `assets/`：存放页面可能依赖的公共资源，不包含外部链接。

## 页面归档说明

- 所有归档页面均为独立的 HTML 文件，可直接在浏览器中打开。
- 页面内容保持原始状态，不进行修改或重构。
- 归档的页面可能包含文本、图片、样式及脚本，但不包含动态数据接口。
- 每个页面文件均以 `page-编号.html` 格式命名，便于索引与引用。

## 维护说明

- 本仓库由维护者不定期更新，添加新的归档页面或调整索引结构。
- 不保证所有页面长期可用或兼容最新浏览器版本。
- 如发现页面显示异常或链接失效，可通过 Issue 反馈，但不保证及时修复。
- 不接受外部提交的页面归档请求，仅由维护者自行决定添加内容。

## 许可

本仓库内容仅供存档与展示之用。各归档页面可能包含各自独立的版权声明，请以页面内标注为准。仓库整体结构及索引文件采用 [MIT 许可证](LICENSE) 发布。
